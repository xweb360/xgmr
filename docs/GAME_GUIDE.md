# 🎮 Neuro-Link: Shadow Proxy 玩家指南

欢迎来到 **Neuro-Link: Shadow Proxy**。这是一款基于 AI 骨骼追踪技术的赛博朋克动作游戏。你将通过身体动作直接控制你的数字代理人（Agent），在虚拟网络空间中粉碎敌人并收集数据。

---

## 🔰 入门篇：基础操作与生存

### 1. 环境准备
*   **光照**：确保你的房间光线充足，以便摄像头能清晰捕捉你的动作。
*   **距离**：站在距离摄像头 1.5 - 2 米处，确保你的全身（至少上半身）都在画面内。
*   **校准**：游戏开始前，保持身体中立位，双臂自然下垂。

### 2. 核心动作控制
*   **移动 (Move)**：身体向左或向右微倾。你的 Agent 会同步在赛道上左右滑动。
*   **攻击 (Attack)**：单手或双手快速向前做“推掌”或“冲拳”动作。
*   **防御 (Defend)**：双手在胸前交叉或并拢握拳。此时 Agent 会发出黄色光芒，进入防御姿态。
*   **子弹时间 (Bullet Time)**：快速下蹲。世界速度会减慢，Agent 发出强烈的白光。

### 3. 基础得分
*   **击碎敌人**：使用攻击动作击碎红色八面体敌人。
*   **收集道具**：直接碰撞绿色能量球或黄色护盾。
*   **维持连击**：连续击碎敌人而不漏掉任何一个，连击数（Combo）会不断攀升，分数加成也会随之提高。

---

## 🚀 进阶篇：高阶技巧与刷分策略

### 1. 动作联动加成 (Action Synergy)
*   **数据块粉碎 (Data Smashing)**：
    *   *普通收集*：直接碰撞紫色数据块仅获得少量分数。
    *   *攻击破坏*：在接触紫色数据块的瞬间执行**攻击动作**，可获得 **10 倍** 积分奖励！
*   **时空收割 (Time Harvest)**：
    *   在**子弹时间（下蹲）**状态下收集绿色能量球，得分将翻倍。利用下蹲减速来精准收割密集的能量球。

### 2. 防御反击 (Defensive Parry)
*   当敌人过多来不及攻击时，迅速进入**防御姿态**。
*   撞击护盾的敌人会被弹开并消灭，虽然得分略低于直接攻击，但它能**保护你的连击不被中断**。

### 3. 难度适应性
*   游戏难度会随着你的分数提升而动态增加（敌人生成速度变快）。
*   **高阶策略**：在高难度下，优先使用“左右倾斜”避开无法处理的敌人，利用“防御”保住高倍率连击，寻找机会攻击紫色数据块获取爆发性得分。

### 4. 视觉残留利用
*   你的动作会留下视觉残留（残影）。这不仅是视觉效果，它能帮助你判断攻击判定的持续时间，掌握出拳的节奏。

---

## 🛠️ 常见问题排查
*   **动作不灵敏？** 检查背景是否过于杂乱，尽量选择纯色背景。
*   **无法获取摄像头？** 确保使用 `http://localhost` 访问，并已授予浏览器摄像头权限。
*   **Agent 乱动？** 尝试重新站位，确保摄像头能看到你的肩膀和手肘关节。
