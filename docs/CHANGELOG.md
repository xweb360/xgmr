## [0.1.0] - 2026-01-20

### 🎉 首次发布

这是 Neuro-Link: Shadow Proxy 的首个公开版本！

### ✨ Added (新增)

#### Phase 1: MVP
- ✅ Vite + TypeScript 项目脚手架
- ✅ Three.js 3D 场景系统
  - 赛博朋克风格灯光
  - 网格地面
  - 雾效果
- ✅ MediaPipe Holistic 集成
  - 全身姿态追踪 (33 点)
  - 手部追踪 (21×2 点)
- ✅ 骨骼可视化系统
  - 实时霓虹节点显示
  - 骨骼连接线
  - 手部追踪可视化
- ✅ 赛博朋克 HUD
  - 加载屏幕
  - 连接状态显示
  - FPS 计数器
- ✅ 游戏主循环
  - 固定时间步长物理更新
  - 可变时间步长逻辑更新
  - 60 FPS 目标

#### Phase 2: 意图引擎
- ✅ SmoothingBuffer 数据平滑系统
  - 5帧移动平均
  - 消除高频抖动
- ✅ IntentEngine 意图识别引擎
  - 侧滑检测 (重心偏移)
  - 子弹时间检测 (下蹲)
  - 防御检测 (双手握拳)
  - 攻击检测 (Z轴推击)
- ✅ 事件总线系统
  - 类型安全的发布-订阅
  - 支持一次性监听
- ✅ 视觉残留特效
  - 意图触发时的骨骼残影
  - 淡出和膨胀动画
  - 故障位移效果

#### Phase 3: 游戏系统
- ✅ Agent 角色系统
  - 程序化 Low-poly 模型
  - 发光材质
  - 动画状态机
  - 意图响应系统
- ✅ Enemy 敌人系统
  - 八面体几何体
  - AI 移动逻辑
  - 死亡动画
- ✅ GameManager 游戏管理器
  - 自动敌人生成 (2秒间隔)
  - 碰撞检测
  - 计分系统
  - 连击系统
- ✅ HUD 增强
  - 分数显示
  - 连击数显示
  - 意图状态显示
  - 冷却条 (预留)

#### Phase 4: 抛光
- ✅ AudioManager 音效系统
  - Web Audio API 程序化音效
  - 侧滑音效 (电子风声)
  - 攻击音效 (低频冲击波)
  - 防御音效 (能量充能)
  - 子弹时间音效 (时间减缓)
  - 命中音效 (白噪声爆裂)
- ✅ ParticleSystem 粒子系统
  - GPU 加速粒子
  - 爆炸效果
  - 重力模拟
  - 淡出动画
- ✅ 完整文档
  - README.md
  - 开发指南
  - 架构文档
  - 贡献指南
  - API 文档

### 🎨 样式系统
- ✅ CSS 变量系统 (设计令牌)
- ✅ 霓虹色调 (青/粉/绿)
- ✅ CSS 动画库
  - 脉冲发光
  - 霓虹闪烁
  - 扫描线
  - 故障效果

### 🛠️ 技术栈
- Vite 7.2.5
- TypeScript 5.9.3
- Three.js 0.160+
- MediaPipe Holistic 0.5+
- Web Audio API

### 📊 性能指标
- FPS: 55-60
- AI 推理: ~12ms
- 内存占用: ~180MB
- 包体积: ~3.2MB
- 首次加载: ~2.5s

### 🔒 隐私与安全
- ✅ 所有处理在客户端本地完成
- ✅ 不上传任何视频数据
- ✅ 不存储视频帧
- ✅ 用户可随时关闭摄像头

## 版本说明

### 语义化版本格式

```
主版本号.次版本号.修订号

- 主版本号: 不兼容的 API 修改
- 次版本号: 向下兼容的功能性新增
- 修订号: 向下兼容的问题修正
```

### 变更类型

- `Added` - 新增功能
- `Changed` - 功能变更
- `Deprecated` - 即将废弃的功能
- `Removed` - 已移除的功能
- `Fixed` - Bug 修复
- `Security` - 安全相关
