## 🎮 玩法核心

### 核心循环

```
┌──────────────────────────────────────────────────────────────────┐
│                        游戏核心循环                               │
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐      │
│   │  玩家   │───►│   AI    │───►│  Agent  │───►│  游戏   │      │
│   │  手势   │    │ 意图引擎 │    │ 执行动作 │    │  反馈   │      │
│   └─────────┘    └─────────┘    └─────────┘    └─────────┘      │
│        │                                              │          │
│        └──────────────────────────────────────────────┘          │
│                         视觉反馈循环                              │
└──────────────────────────────────────────────────────────────────┘
```

### 玩法描述

玩家通过摄像头做出手势，AI Agent 理解意图后执行华丽的战术动作。

**核心好玩点:** 利用 AI 弥补玩家肢体动作的笨拙，通过**"意图放大"**产生超人快感。

## 🎯 战术动作系统

### 动作列表

| 动作名称 | 触发手势 | Agent 执行效果 | 视觉特效 |
|----------|----------|----------------|----------|
| **侧滑 (Dash)** | 身体重心左/右倾斜 | 高速横向位移 | 残影拖尾 + 地面滑痕 |
| **护盾 (Shield)** | 双手握拳 | 能量护盾展开 | 六边形力场 + 粒子散射 |
| **脉冲攻击 (Pulse)** | 手掌推击 (Z轴加速度) | 能量波发射 | 冲击波纹 + 目标命中特效 |
| **子弹时间 (Bullet Time)** | 下蹲 | 时间减缓 + 闪避 | 全局灰度 + 拖影效果 |

### 动作优先级

```
子弹时间 (最高) > 护盾 > 脉冲攻击 > 侧滑 (最低)
```

### 动作冷却

| 动作 | 冷却时间 | 持续时间 |
|------|----------|----------|
| 侧滑 | 0.3s | 0.2s |
| 护盾 | 2.0s | 1.5s (可提前取消) |
| 脉冲攻击 | 0.5s | 0.3s |
| 子弹时间 | 5.0s | 2.0s |

## 🖥️ 视觉规范

### Meta-data Visuals (元数据可视化)

实时显示 AI 追踪的骨骼节点，增强赛博朋克氛围：

1. **骨骼节点** - 发光圆点标记关键关节
2. **连接线** - 霓虹色线条连接骨骼
3. **置信度** - 节点亮度反映追踪置信度
4. **运动轨迹** - 高速运动时显示轨迹残留

### 配色方案

| 元素 | 颜色代码 | 用途 |
|------|----------|------|
| **主色 - 霓虹青** | `#00FFFF` | 骨骼节点、UI 高亮 |
| **辅色 - 霓虹粉** | `#FF00FF` | 攻击特效、警告 |
| **强调 - 霓虹绿** | `#00FF00` | 确认、成功反馈 |
| **背景 - 深空灰** | `#0A0A0F` | 主背景色 |
| **次背景** | `#1A1A2E` | 面板背景 |

### 特效规格

| 特效 | 渲染方式 | 性能预算 |
|------|----------|----------|
| 残影拖尾 | Shader + 多实例 | < 2ms/帧 |
| 力场护盾 | 自定义 Shader | < 1ms/帧 |
| 脉冲波 | 后处理特效 | < 2ms/帧 |
| 粒子系统 | GPU 粒子 | < 1000 粒子 |
